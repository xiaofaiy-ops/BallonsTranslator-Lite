# BallonsTranslator Lite - 简化版漫画翻译工具

## 项目概述
- **项目名称**: BallonsTranslator Lite
- **类型**: 桌面应用程序 (PyQt6)
- **核心功能**: 一键漫画翻译，去除复杂编辑界面，保留核心翻译功能
- **目标用户**: 需要快速翻译漫画的非专业用户

## UI/UX 规格

### 窗口配置
- **窗口尺寸**: 800x600 (最小), 可调整大小
- **窗口标题**: BallonsTranslator Lite
- **窗口样式**: 现代化简洁风格

### 布局结构
```
┌─────────────────────────────────────────┐
│  顶部工具栏 (语言选择 + 翻译按钮)        │
├─────────────────────────────────────────┤
│                                         │
│           图片拖放区域                   │
│        (支持拖放或点击选择)              │
│                                         │
├─────────────────────────────────────────┤
│  状态栏 (翻译进度 / 状态信息)            │
└─────────────────────────────────────────┘
```

### 视觉设计
- **配色方案**:
  - 主色: #2C3E50 (深蓝灰)
  - 强调色: #3498DB (蓝色)
  - 背景色: #ECF0F1 (浅灰)
  - 文字色: #2C3E50
- **字体**: 系统默认字体, 12px
- **间距**: 16px 边距, 8px 元素间距

### 组件
1. **语言选择下拉框**
   - 源语言: 日语/英语/中文/韩语
   - 目标语言: 中文/英语/日语/韩语
   
2. **翻译按钮**
   - 状态: 默认 / 翻译中 (禁用) / 完成
   
3. **拖放区域**
   - 默认显示提示文字和图标
   - 拖入图片后显示缩略图
   
4. **进度条**
   - 显示翻译进度百分比

## 功能规格

### 核心功能
1. **图片导入**
   - 支持拖放 JPG/PNG/WEBP/GIF
   - 支持点击选择文件
   - 支持文件夹批量导入

2. **一键翻译**
   - 自动检测文字
   - OCR 识别
   - 机器翻译
   - 文字回填渲染

3. **结果保存**
   - 自动保存到输出目录
   - 命名规则: 原文件名_translated.png

### 数据流
```
图片输入 → 文字检测 → OCR识别 → 翻译 → 文字渲染 → 输出图片
```

### 配置
- 默认翻译器: Google Translate (免费)
- 可配置 DeepL/Sugoi 等
- 输出目录: 与原图同目录
